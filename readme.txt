      - Some brains

  ## Configuration
    Ok, you need edit defines inside config.php file.

    DEFINE                | ???
    ---------------------------
    ESN_TIMEZONE_SET      | Default timezone (need to show time)
    ESN_MYSQL_UNAME       | MySQL Username
    ESN_MYSQL_PWRD        | MySQL Password
    ESN_MYSQL_SERVER      | MySQL Server
    ESN_MYSQL_DB          | MySQL Database
    ESN_SITE_NAME         | Site name
    ESN_SITE_SLOGAN       | Site description
    ESN_SITE_DOMAIN       | Domain
    ESN_SITE_COPYRIGHT    | Copyright info
    ESN_BOOL_SHOW_SLOGAN  | Show description or not? (TRUE/FALSE)
